# Poker CFR Solver

Counterfactual Regret Minimization for solving imperfect-information poker games (Kuhn, Leduc).

## Quickstart

```bash
git clone https://github.com/abhi-wadhwa/poker-cfr-solver.git
cd poker-cfr-solver
make install
make run
make test
```
